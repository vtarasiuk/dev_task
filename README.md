# dev_task

## Install

## Requirements

- CMake 3.22.1 or newer [CMake Download](https://cmake.org/download/)
- C compiler (for example, GCC)

### Building a project

1.  ```sh
    git clone https://github.com/vtarasiuk/dev_task.git
    cd devtask
    ```

2.  ```sh
    mkdir build
    cmake -B build
    ```

3.  ```sh
    cmake --build build --config Release
    ```
